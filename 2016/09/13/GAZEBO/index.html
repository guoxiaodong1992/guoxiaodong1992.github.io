四旋翼

基于PX4和Gazebo的飞行仿真

实验背景
小型四旋翼无人机具有成本低，体积小，机械结构简单，灵活性高等优势，被逐步运用到生产生活中的诸多领域，如电力线路巡检、航拍、农业植保、灾情勘察、地理测绘等。而在无人机的飞控实现完成后，要想使其完成复杂的自主飞行任务，需要设计控制算法对无人机的飞行和任务执行进行控制。而算法的仿真实现对算法的设计和验证都有重要的意义。相比实际飞行，仿真飞行减少了时间成本和物质成本，同时方便对算法的逻辑和参数进行重复修改。本文利用Gazebo机器人仿真平台和PX4飞行固件对无人机飞行进行软件仿真。利用PX4的Offboard Control模式，对其进行基于GPS的和基于雷达的飞行控制。

PX4介绍
PX4是一款软件和硬件同时开源飞控项目。该项目起源于瑞士苏黎世联邦理工大学的计算机视觉与几何实验室的PIXHAWK项目，是目前最流行的开源飞控之一。PX4FMU自驾仪模块运行高效的(RTOS)实时ARM操作系统，其固件同时附带了一系列工具集、系统驱动/模块与外围软件接口层，所有这些软件（包括用户自定义的飞控软件）随OS内核一起，统一编译为固件形式，然后上传到飞控板中，从而实现对飞控板的软件配置。
PX4配套的软件架构主要分为4层：
a) API层：提供给用户做开发的接口层。
b) 框架层：包含了操作基础飞行控制的默认程序集（节点）
c) 系统库：包含了所有的系统库和基本交通控制的函数
d) OS内核：提供硬件驱动程序、网络、UAVCAN和故障安全系统
以上是面向PX4系统实现者的具体的软件架构。如果从飞控应用开发者的角度对软件架构进行描述，可以将PX4软件分为两层：
a)PX4 Flight Stack: 自主无人机控制算法的集合。
b)PX4 Middleware: 针对无人机控制器，传感器等物理设备的驱动及底层通信、调度等机制的集合。
多态性（抽象性）是PX4软件架构最显著的特点之一。为了最大限度地保障飞控算法代码的重用性，其将飞控逻辑与底层控制器的指令实现进行了解耦合。利用同一套高层的飞控算法，即可适用于固定翼、直升机、多旋翼等多种机型的控制场合。

MAVLINK飞行控制协议介绍
MAVLINK( Micro Air Vehicle Communication Protocol) 协议，是一种在串口通讯基础上的更高层的开源通信协议，主要用于微小型无人机的通信，包括无人机与地面站以及无人机和机载电脑的通信。该协议于2009年由瑞士苏黎世联邦理工大学的计算机视觉与几何实验室的Lorenz Meier，并遵循LGPL开源协议。PX4提供了对MAVLINK协议的原生支持。
MAVLINK协议传输的基本单位是消息帧。下图描述了消息帧的基本单位。除了灰色格子，其他的格子代表了一个字节的数据。

STX代表起始标志位，在1.0版本中，起始标志位是“FE”。该标志位用于MAVLINK消息帧的消息解码。
LEN代表消息帧中PAYLOAD（有效载荷）的字节长度。范围为0~255。在收到一帧消息之后，接收端可以用它和实际收到的有效载荷的字节长度进行比较，以验证有效载荷部分长度的正确性。
SEQ代表的是本次消息帧的序号。每发完一个消息，这个字节的内容会加1，在255之后会从0重新开始。接收端用本子节的信息计算消息丢失的比例。
SYS代表的是发送本次消息帧的设备系统编号。使用PX4固件的PIXHAWK设备发送消息时系统编号是1，该字节用于接收端识别发送端的设备类型。
COMP代表的是发送本条消息帧的设备的单元编号。使用PX4固件的PIXHAWK设备发送消息时默认的单元编号是50，该字节用于接收端识别书发送设备的哪个单元发来的消息。
MSG代表有效载荷中消息包的类型编号。接收端根据这个编号来确定有效载荷中的消息类型，从而进行对有效载荷部分的信息的正确解码。
PAYLOAD 是消息帧最重要的部分，其中包含了有效载荷的信息。不同消息类型的PAYLOAD字节数长度不一定相同。但都在0-255之间。
CKB和CKA是校验位。CKB是高8位校验位，CKA是第八位校验位。该校验码有CRC16算法得到，该算法将消息帧的起始位开始到有效载荷的结束部分进行CRC16计算，从而得到16位的检验码。接收端通过该16位的校验码对收到的消息帧进行校验。
以MSG编号为33的消息帧为例，分析消息帧的PAYLOAD组成。该帧消息用来传递无人机当前的GPS位置信息。由下图可知，该消息一共由9部分组成，PAYLOAD的长度为28个字节。消息的第1部分time_boot_ms表示发送消息时系统的时间戳。第2部分lat表示飞机的纬度值，该值是由真实的纬度值乘10的7次方得到的整型值。lon表示飞机经度值。alt表示飞机的海拔高度。而relative_alt表示飞机相对地面的高度。vx,vy,vz分别表示飞机在NED坐标下各分量的速度，以真实值乘100得到的整型值表示。hdg代表飞机当前的yaw角值，以真实值乘100得到的整型值表示。

再以MSG编号为76的COMMAND_LONG消息帧为例，分析地面站或者机载电脑如何通过MAVLINK给无人机发送控制指令。下图描述了该消息帧的PAYLOAD组成。该消息的PAYLOAD由11部分组成，共有33个字节。target_system表示目标的系统编号。target_component表示目标的单元编号。第三部分command非常重要，它描述了该条指令的任务类型，如起飞，降落等等。所有的任务类型在MAV_CMD中定义。不同任务类型的参数个数不一样，但最多有7个参数。

以在GPS导航模式下的起飞任务为例，此时command参数为MAV_CMD_NAV_TAKEOFF, 该任务模式下的参数组成如下图所示。参数1,2,3都没有具体意义。而参数4,5,6分别代表期望的角度，纬度，经度和高度。

仿真程序分析
Gazebo是一款3D的机器人仿真软件。利用Gazebo进行飞行仿真的程序框图如下图所示：其中SITL指的是Software in the Loop Simulation，即软件仿真。与其相对的是HIL，Hardware in the Loop Simulation, 即硬件仿真。本实验采用的是SITL。

Plugin是Gazebo提供给开发者的方针模型控制接口，开发者可以自主定义仿真中用到的模型的类型以及参数，并且可以利用自己的程序控制它们。本实验中主要利用了三种类型的Plugin，包括World, Model, 以及Sensor。
MAVROS是一个ROS程序包，它的作用就是把MAVLINK消息转换成ROS中的消息，这样就可以以话题的形式在ROS中接受订阅和发布，方便基于ROS的程序控制。
利用PX4的Offboard Control模式可以实现利用电脑控制无人机飞行。使用这个模式尤其需要注意的是，在开启该模式后，电脑发送给飞控板的控制指令频率不能小于2 Hz,否则飞控会自动切离该模式。
实验分为两个阶段。第1个阶段为起飞阶段。步骤为：切到Offboard Control模式，解锁，设置GPS目标起飞高度,程序中设定的目标起飞高度为1.2m。实验的第2个阶段是以恒定的速度通过走廊，并利用雷达的数据检测墙上的门，在检测到墙上的门之后，改变飞行方向，进入门内。

仿真结果
在本次实验中，成功在单机上实现了利用GPS和雷达数据的Gazebo飞行仿真。在下一步的仿真实验中，将重点研究多无人机在同一仿真场景下的飞行控制，重点在有搭建多条通讯链路和Gazebo的控制模型，从而实现对多架无人机的同步控制，为多无人机编队算法和相应的避障算法的设计和验证提供仿真依据。

附2：ros主程序代码

#include <ros/ros.h>
#include <offb_demo/offb_node.h>


int main(int argc, char **argv)
{
    ros::init(argc, argv, "offb_node");
    ros::NodeHandle nh;
    PX4Drone* drone =new PX4Drone(nh);
    ROS_INFO("new 0904 PX4 OFFBOARD CONTROL TEST");
    //the setpoint publishing rate MUST be faster than 2Hz
    ros::Rate rate(40.0);


    //Init Para
    drone->global_set_vel.twist.linear.x=0;
    drone->global_set_vel.twist.linear.y=0;
    drone->global_set_vel.twist.linear.z=0.2;




    // wait for FCU connection
    while(ros::ok() && drone->current_state.connected){
        ros::spinOnce();
        rate.sleep();
    }



    //send a few setpoints before starting
    for(int i = 100; ros::ok() && i > 0; --i){
        drone->velocity_pub.publish(drone->global_set_vel);
        ros::spinOnce();
        rate.sleep();
    }




    drone->offb_set_mode.request.custom_mode = "OFFBOARD";
    drone->arm_cmd.request.value = true;


    ros::Time last_request = ros::Time::now();

    while(ros::ok())
    {
        ros::spinOnce();
        rate.sleep();



        drone->velocity_pub.publish(drone->global_set_vel);
        //drone->global_pos_pub.publish(drone->ideal_global_pos);
        if(drone->task_complete_flag==1)
        {
           switch(drone->task_update_flag)
           {
             case 0:
            if( drone->current_state.mode != "OFFBOARD" &&
                (ros::Time::now() - last_request > ros::Duration(5.0))){
                if( drone->set_mode_client.call(drone->offb_set_mode) &&
                    drone->offb_set_mode.response.success){
                    ROS_INFO("Offboard enabled");
                }
                last_request = ros::Time::now();
            } else {
                if( !drone->current_state.armed &&
                    (ros::Time::now() - last_request > ros::Duration(5.0))){
                    if( drone->arming_client.call(drone->arm_cmd) &&
                        drone->arm_cmd.response.success){
                        ROS_INFO("Vehicle armed");
                    }
                    last_request = ros::Time::now();
                }
            }

            if(drone->current_state.armed)
            {
              drone->initial_global_pos=drone->global_pos;
              drone->ideal_global_pos.latitude=drone->initial_global_pos.latitude;
              drone->ideal_global_pos.longitude=drone->initial_global_pos.longitude;
              drone->ideal_global_pos.altitude=drone->initial_global_pos.altitude+1.2;
              std::cout<<"The ideal latitude is: "<<drone->ideal_global_pos.latitude<<std::endl;
              drone->task_update_flag=1;
              drone->task_flight_mode=1;
            }
            break;

            case 1:
               drone->task_complete_flag=0;
               break;

           case 2:
                  drone->task_complete_flag=0;
                  break;
           case 3:
                   drone->task_complete_flag=0;
                    break;
          case 4:
                    drone->task_complete_flag=0;
                      break;



            default:
               break;

          // local_pos_pub.publish(pose);
            // drone->velocity_pub.publish(drone->global_set_vel);

            //  std::cout<<"The velocity of X is: "<<drone->local_vel.twist.linear.x<<std::endl;
            //  std::cout<<"The Latitude is: "<<drone->global_pos.latitude<<std::endl;
      }
    }
    }

    return 0;
}

附3：ros头文件代码

#include <ros/ros.h>
#include <cmath>
#include <float.h>
#include <iostream>
#include <geometry_msgs/PoseStamped.h>
#include <geometry_msgs/TwistStamped.h>
#include <mavros_msgs/CommandBool.h>
#include <mavros_msgs/CommandTOL.h>
#include <mavros_msgs/SetMode.h>
#include <mavros_msgs/State.h>
#include <std_msgs/Float64.h>
#include <sensor_msgs/NavSatFix.h>
#include <mavros_msgs/GlobalPositionTarget.h>
#include <mavros_msgs/OpticalFlowRad.h>


#define CONSTANTS_RADIUS_OF_EARTH 6371000 /*meters(m)*/
#define CONSTANTS_DEG_TO_RAD 0.01745329251994

class PX4Drone
{
private:

  //SUbscriber
      ros::Subscriber state_sub;
      ros::Subscriber global_vel_sub;
      ros::Subscriber local_vel_sub;
      ros::Subscriber global_pos_sub;
      ros::Subscriber flow_came_sub;


public:
  //Publisher
      ros::Publisher local_pos_pub;
      ros::Publisher velocity_pub;
      ros::Publisher global_pos_pub;
    //ServiceClient
      ros::ServiceClient arming_client;
      ros::ServiceClient set_mode_client;
      ros::ServiceClient takeoff_client;
      ros::ServiceClient land_client;
      mavros_msgs::CommandBool arm_cmd;
      geometry_msgs::TwistStamped global_vel;
      geometry_msgs::TwistStamped local_vel;
      geometry_msgs::TwistStamped global_set_vel;

      sensor_msgs::NavSatFix  global_pos;
      sensor_msgs::NavSatFix  local_pos;
      mavros_msgs::State current_state;
      mavros_msgs::SetMode offb_set_mode;
      mavros_msgs::OpticalFlowRad optical_flow;

      //Parameter
      bool task_complete_flag;
      int  task_update_flag;
      int  task_flight_mode;
      float err_x,err_y,err_z;
      float ctrl_x,ctrl_y,ctrl_z,ctrl_max;
      float dist_x,dist_y;
      sensor_msgs::NavSatFix  initial_global_pos;
      sensor_msgs::NavSatFix ideal_global_pos;

private:

      void cal_distance()
      {
        double lat_rad=global_pos.latitude*CONSTANTS_DEG_TO_RAD;
        double lon_rad=global_pos.longitude*CONSTANTS_DEG_TO_RAD;
        double ideal_lat_rad=ideal_global_pos.latitude*CONSTANTS_DEG_TO_RAD;
        double ideal_lon_rad=ideal_global_pos.longitude*CONSTANTS_DEG_TO_RAD;

        double sin_lat=sin(lat_rad);
        double cos_lat=cos(lat_rad);
        double sin_ideal_lat=sin(ideal_lat_rad);
        double cos_ideal_lat=cos(ideal_lat_rad);
        double cos_d_lon=cos(lon_rad-ideal_lon_rad);

        double arg=sin_ideal_lat*sin_lat+cos_ideal_lat*cos_lat*cos_d_lon;
        if(arg>1.0)
        arg=1.0;
        else if(arg<-1.0)
        arg=-1.0;
        double c=acos(arg);
        double k = (fabs(c) < DBL_EPSILON) ? 1.0 : (c / sin(c));

       dist_y=k*(cos_ideal_lat*sin_lat-sin_ideal_lat*cos_lat*cos_d_lon)*CONSTANTS_RADIUS_OF_EARTH;
       dist_x=k*cos_lat*sin(lon_rad-ideal_lon_rad)*CONSTANTS_RADIUS_OF_EARTH;

      }
      void state_callback(const mavros_msgs::State::ConstPtr& msg){
          this->current_state = *msg;
      }

      void global_vel_callback(const geometry_msgs::TwistStamped msg){
          this->global_vel = msg;
      }

      void local_vel_callback(const geometry_msgs::TwistStamped msg){
          this->local_vel = msg;
      }

      void global_pos_callback(const sensor_msgs::NavSatFix msg)
      {
        this->global_pos=msg;
        this->cal_distance();
        this->err_x=-1.0*this->dist_x;
        this->err_y=-1.0*this->dist_y;
        this->err_z=this->ideal_global_pos.altitude-this->global_pos.altitude;
        this->ctrl_x=this->err_x;
        this->ctrl_x=((this->ctrl_x)<(-1.0*ctrl_max)?(-1.0*ctrl_max):(this->ctrl_x)>ctrl_max?ctrl_max:this->ctrl_x);
        this->ctrl_y=this->err_y;
        this->ctrl_y=((this->ctrl_y)<(-1.0*ctrl_max)?(-1.0*ctrl_max):(this->ctrl_y)>ctrl_max?ctrl_max:this->ctrl_y);
        this->ctrl_z=this->err_z;
        this->ctrl_z=((this->ctrl_z)<(-1.0*ctrl_max)?(-1.0*ctrl_max):(this->ctrl_z)>ctrl_max?ctrl_max:this->ctrl_z);
        if(this->task_flight_mode==1)
        {
          this->global_set_vel.twist.linear.x=this->ctrl_x;
          this->global_set_vel.twist.linear.y=this->ctrl_y;
          this->global_set_vel.twist.linear.z=this->ctrl_z;
          std::cout<<"The control value x and y and z is: "<<this->ctrl_x
                   <<" and "<<this->ctrl_y
                   <<" and "<<this->ctrl_z<<std::endl;
          if(fabs(this->ctrl_z)<0.2)
          {
            if(!this->task_complete_flag)
            {
                this->task_update_flag++;
                this->task_complete_flag=1;
                this->task_flight_mode++;
            }

          }
        }

      }


      void flow_came_callback(const mavros_msgs::OpticalFlowRad msg)
      {
          this->optical_flow = msg;

          if(this->task_flight_mode==2&&this->optical_flow.distance>3.0)
          {
            if(!this->task_complete_flag)
            {
                this->task_update_flag++;
                this->task_complete_flag=1;
                this->task_flight_mode++;
            }

          }


          if(this->task_flight_mode==3&&this->optical_flow.distance<3.0)
          {
            if(!this->task_complete_flag)
            {
                this->task_update_flag++;
                this->task_complete_flag=1;
                this->task_flight_mode++;
            }

          }


          if(this->task_flight_mode==2)
          {
          this->global_set_vel.twist.linear.x=0;
          this->global_set_vel.twist.linear.y=-0.3;
          this->global_set_vel.twist.linear.z=this->ctrl_z;
        }

        if(this->task_flight_mode==3)
        {
        this->global_set_vel.twist.linear.x=0.3;
        this->global_set_vel.twist.linear.y=-0.08;
        this->global_set_vel.twist.linear.z=this->ctrl_z;
      }

      if(this->task_flight_mode==4)
      {
      this->global_set_vel.twist.linear.x=0;
      this->global_set_vel.twist.linear.y=0;
      this->global_set_vel.twist.linear.z=this->ctrl_z;
    }

      }


public:

       PX4Drone(ros::NodeHandle& nh)
       {
         state_sub = nh.subscribe<mavros_msgs::State>
                 ("mavros/state", 10, &PX4Drone::state_callback,this);
        global_vel_sub = nh.subscribe<geometry_msgs::TwistStamped>
                         ("mavros/global_position/raw/gp_vel", 10, &PX4Drone::global_vel_callback,this);
        flow_came_sub = nh.subscribe<mavros_msgs::OpticalFlowRad>
                          ("mavros/px4flow/raw/optical_flow_rad", 10, &PX4Drone::flow_came_callback,this);
          local_vel_sub = nh.subscribe<geometry_msgs::TwistStamped>
                         ("mavros/local_position/velocity", 10, &PX4Drone::local_vel_callback,this);
         global_pos_sub = nh.subscribe<sensor_msgs::NavSatFix>
                         ("mavros/global_position/global", 10, &PX4Drone::global_pos_callback,this);
         local_pos_pub = nh.advertise<geometry_msgs::PoseStamped>
                 ("mavros/setpoint_position/local", 10);
         global_pos_pub=nh.advertise<mavros_msgs::GlobalPositionTarget>
                  ("mavros/setpoint_raw/global",10);
          velocity_pub = nh.advertise<geometry_msgs::TwistStamped>
                  ("mavros/setpoint_velocity/cmd_vel", 10);
          arming_client = nh.serviceClient<mavros_msgs::CommandBool>
                 ("mavros/cmd/arming");
          set_mode_client = nh.serviceClient<mavros_msgs::SetMode>
                 ("mavros/set_mode");
          takeoff_client=nh.serviceClient<mavros_msgs::CommandTOL>
                 ("mavros/cmd/takeoff");
          land_client=nh.serviceClient<mavros_msgs::CommandTOL>
                        ("mavros/cmd/land");

          //Parameter init

          task_complete_flag=1;
          task_flight_mode=0;
          task_update_flag=0;
          ctrl_max=0.3;

       }
       bool takeoff()
       {
         mavros_msgs::CommandTOL drone_commandtol;
         drone_commandtol.request.min_pitch=0;
         drone_commandtol.request.latitude=this->global_pos.latitude;
         drone_commandtol.request.longitude=this->global_pos.longitude;
         drone_commandtol.request.yaw=0;
         std::cout<<"Latitude: "<<this->global_pos.latitude<<std::endl;
         std::cout<<"Longitude: "<<this->global_pos.longitude<<std::endl;
         drone_commandtol.request.altitude=this->global_pos.altitude+3.0;
         std::cout<<"Altitude: : "<<drone_commandtol.request.altitude<<std::endl;
         return takeoff_client.call(drone_commandtol)&&drone_commandtol.response.success;
       }
       bool land()
       {
         mavros_msgs::CommandTOL drone_commandtol;
         return land_client.call(drone_commandtol)&&drone_commandtol.response.success;
       }
};
